# ESP32 Web Server

A modular web server implementation for ESP32 microcontrollers using ESP-IDF.

## Features

- Wi-Fi Access Point (AP) mode
- Configurable SSID and password via menuconfig
- HTTP server with request handlers
- Modular component architecture
